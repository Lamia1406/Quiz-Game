# Knowledge Quiz Game

This project is a mini-application designed to demonstrate the capabilities of a Question-Answer system using the principles of the Web of Data and SPARQL queries.

## Features
- User-Friendly Interface: A simple homepage where users can select from a set of predefined questions.
- Dynamic SPARQL Queries: Formulation of diverse SPARQL queries for each predefined question, incorporating features like OPTIONAL, FILTER, ORDER BY, LIMIT, OFFSET, GROUP BY, UNION, and various operators.
- DBpedia Integration: Seamless integration with the DBpedia SPARQL endpoint (http://dbpedia.org/sparql) to retrieve information in a specified format.

## Installation

To get started with the Internship Management Application, follow the steps below:

1. Clone the repository(Or download in .zip format)

* $ git clone https://github.com/Lamia1406/Quiz-Game.git
* $ cd Quiz-Game

2. Install dependencies:
* $ npm install

3. Start the client development server
* $ npm start

4. Access the application:
Open a web browser and visit http://localhost:3000 to access the Quiz-Game Application.
Note: If port 3000 is already in use by another application, React may prompt you with options to switch to an alternative port. Make sure to follow the instructions provided in the terminal if you need to use a different port.
